# Аналитика и прогнозирование популярности русскоязычных стримеров Twitch

Выпускная работа состоит из двух частей:

- **Часть 1** – Исследовательский анализ данных топ-1000 стримеров Twitch (май 2024)
- **Часть 2** – Прогнозная модель: предсказание среднего онлайна стримера на основе метрик канала

## Используемые датасеты
1. `Данные о стримерах Twitch (топ-1000, май 2024).csv` – исходные сырые данные
2. `Прогноз среднего онлайна русскоязычных стримеров на основе показателей канала.csv` – очищенные данные для модели

Источник данных: [Kaggle – Top 1000 Twitch Streamers (May 2024)](https://www.kaggle.com/datasets/hibrahimag1/top-1000-twitch-streamers-data-may-2024)
